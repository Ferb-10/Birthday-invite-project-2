# Birthday-invite-project-2

## Overview
This project is a simple HTML webpage created as a birthday invitation.
It displays the birthday date, an image, a list of items to bring, and a link to the event location using Google Maps.
The main purpose of this project is to practice basic HTML structure and elements.

![image alt](https://github.com/Ferb-10/Birthday-invite-project-2/blob/7ea2e3a68a0fe9b817ebb1ea99f5b7647fd8dfab/birthday-project.png)

## Features
- Displays a birthday announcement with headings
- Shows an image using the <img> tag
- Uses an unordered list to display items to bring
- Includes a Google Maps link for the event location
- Simple and clean static layout

## Technologies Used
- HTML5

## What I Learned
- Basic HTML document structure
- Proper use of semantic elements such as headings, lists, images, and links
- How to link external resources (Google Maps)
- Structuring content in a clear and readable way

## 概要
このプロジェクトは、誕生日パーティーの招待状として作成したシンプルなHTMLページです。
誕生日の日付、画像、持ち物リスト、そしてGoogleマップへのリンクを表示しています。
HTMLの基本構造と要素の使い方を学ぶことを目的とした練習プロジェクトです。

## 機能
- 見出しタグを使った誕生日メッセージの表示

- 持ち物をリスト形式で表示
- イベント場所を示すGoogleマップへのリンク
- シンプルで見やすい静的レイアウト

## 使用技術
- HTML5

## 学んだこと
- HTMLの基本的なドキュメント構造
- 見出し、リスト、画像、リンクなどの適切な使い方
- 外部リンク（Googleマップ）の設定方法
- 情報を整理して分かりやすく表示する方法
